# 🎓 Student Management System

A CLI-based Student Management System built with Python, demonstrating OOP principles and file I/O.

## 📋 Features
- ✅ Add / Update / Delete / List students
- ✅ Unique Student ID validation
- ✅ Data persisted in `students.json`
- ✅ Clean formatted console output
- ✅ OOP design with `Student` and `StudentManager` classes

## 🗂️ Project Structure
```
student-management-system/
├── student.py      # Student class (OOP model)
├── manager.py      # StudentManager class (CRUD + file I/O)
├── main.py         # CLI entry point
├── students.json   # Persistent data store
└── README.md
```

## 🚀 How to Run

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/student-management-system.git
cd student-management-system

# Run the app
python main.py
```

## 🖥️ Usage

| Option | Action              |
|--------|---------------------|
| 1      | Add a student       |
| 2      | Update a student    |
| 3      | Delete a student    |
| 4      | List all students   |
| 5      | Find student by ID  |
| 6      | Exit                |

## 🛠️ Technologies
- Python 3.x
- JSON for data persistence
- OOP (classes, constructors, methods)
